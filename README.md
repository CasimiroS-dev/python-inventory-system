# Python Inventory Management System

A command line inventory management system built in Python.

## Features
- View all products with price and quantity
- Add new products
- Restock existing products
- Sell products with low stock warnings
- Remove products

## Built With
- Python 3
- Core concepts: dictionaries, nested data structures, 
  functions, loops, and error handling

## How to Run
Run inventory.py in any Python 3 environment.

## What I Learned
This project was built while relearning Python fundamentals
including nested dictionaries, function design, input 
validation, and defensive programming patterns.
